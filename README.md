# chore-helper

Create the basic page structure for the todo app with a header, main content section, and footer.

In the header, display "My Todo App". 

In the main content section, add a heading that says "Todos".

Style the page with a clean, modern design. Use a white background, black text, and accent colors like blue for buttons.
